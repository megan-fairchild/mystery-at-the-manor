# Mystery at the Manor

An annual outdoor summer movie night themed around the 1985 film *Clue*. A multi-page event planning website with themed meals, activities, and printable game materials.

## Pages

- **index.html** - Animated invitation page
- **event-hub.html** - Navigation hub with quick reference
- **itinerary.html** - Hour-by-hour timeline
- **activities.html** - Karaoke, whodunit game, movie details
- **meals.html** - Themed recipes with swap options and interactive grocery list
- **clothing.html** - Character color guide and packing checklist
- **printables/** - Printable clue cards and detective certificate

## Theme

Dark mystery aesthetic with jewel-tone accents inspired by the Clue board game characters. Responsive, mobile-first design with print-friendly styles.

## Deployment

This site is static HTML/CSS/JS. Deploy via GitHub Pages on the `main` branch.
